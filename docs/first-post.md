# Phez is Alive

A personal AI agent, built from scratch on an M3 Mac, accessed from an iPhone over SSH.

No cloud platforms. No third-party integrations. No attack surface beyond a Tailscale tunnel.

## What's running

- An always-on MacBook Pro with the lid closed
- Claude Code CLI as the brain
- tmux keeping sessions alive across disconnects
- Markdown files as memory — nothing fancier needed

## What's next

- An X/Twitter skill that pulls and analyzes the timeline
- Cron jobs that run skills on a schedule
- Every output publishable to this site with one command

## Why

Because the best AI agent is one you built yourself, understand completely, and control entirely.
